Use it in your browser

No install, no API key. Set up a reusable assistant in Claude, ChatGPT, or Gemini using your resume and writing samples — it builds a voice fingerprint from them.

1. Pick your platform & set up the project

  1. Go to claude.ai and log in.
  2. Click Projects → New Project; name it "Job Applications".
  3. Under Files, upload your resume, 2–4 writing samples, and (optional) your LinkedIn PDF.
  4. Under Project Instructions, paste the block below.
  5. Start a new chat and paste the Kickoff (or Interview-Prep) message below.

  5. Start a new chat in the notebook and paste the Kickoff (or Interview-Prep) message below.

Download the app

Prefer everything on your own machine? Run the self-hosted app with your own API key, a logged-in CLI, or a local Ollama model. Your data never leaves your computer.

  1. Download the project from GitHub (green Code button → Download ZIP, then unzip).
  2. Run the start script for your OS: start-mac.command (macOS), start-windows.bat (Windows), or start.sh (Linux). It opens the app in your browser.
  3. Click Connect an LLM and choose an API key, CLI login, or local Ollama model.
